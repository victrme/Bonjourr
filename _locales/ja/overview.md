Bonjourr は Momentum や Tabliss といった拡張機能の現代的な代替品です — iOS
にインスパイアされたデザインと強力なカスタマイズ機能を組み合わせ、思わず開きたくなる、落ち着いた使いやすい新しいタブページを作り出します。

🏆 2023年 Google Chrome おすすめ拡張機能に選出

🌄 デフォルトから美しく

- 時間帯に合わせて雰囲気が変わるダイナミックな4Kと動画の壁紙
- すっきりとした iOS スタイルのインターフェース
- 軽快でなめらかな動作

⚡️ 生産性ツール

- クイックリンクでブックマークを管理
- カスタム検索エンジンを含む全検索エンジン対応の検索バー
- Markdown によるメモとチェックリスト
- ポモドーロタイマー
- 天気予報
- 毎日のインスピレーション名言

🎨 自由なカスタマイズ

- 壁紙・ブックマーク・レイアウトを細かく調整
- 複数スタイルのアナログ時計
- カスタムフォントと CSS スタイル
- ダークモード
- 柔軟な外観オプション
- 43言語対応

🔒 プライバシーファースト Bonjourr
はデータを一切収集せず、アカウント不要で、必要最低限の権限のみ使用します。壁紙はすべて人の手で選ばれたもので、生成AIは使用していません。完全オープンソースで透明性を大切にしています。

👨‍💻 独立開発 Bonjourr
は、デザイン・プライバシー・丁寧な仕事づくりを心から大切にする2人の独立した開発者によって作られています —
広告主や投資家の都合で方向性が決まる会社ではありません。

美しいホームページや新しいタブページをお探しなら、Bonjourr はあなたのために作られました。
